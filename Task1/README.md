## <a name="overview">Overview</a>
It is a Tkinter-based Python application built with Object-Oriented Programming principles, serving as a comprehensive Library Management System.

## <a name="coreFunctions">Core Functions</a>
**User Management**
- Account registration
- Change password
- View personal borrowing records

**Book Management**
- Add new books to catalog
- Remove books from catalog
- Search for books by title/author/ISBN

**Borrowing System**
- Borrow available books
- Return borrowed books
- Automatic fine calculation for overdue returns

## <a name="roles">User roles</a>
**Public User:**
- Register/login
- Change password
- View borrowing history
- Borrow/return books

**Admin:**
- All public user functions
- Add/remove books
- Search books
- View all users' records

## <a name="modules">Modules</a>
- main.py - Entry point of application
- gui.py - Tkinter graphical user interface
- User.py - Manages user authentication and data
- Book.py - Manages book catalog and search
- Library.py - Manages borrowing, returning and fines for due
